SVG Icon Web Component

Lightweight, framework-agnostic SVG icons as Web Components

Features

Framework Agnostic

Use icons as Web Components across any framework (Angular, React, Vue, plain JS)

Dynamic Loading

Dynamically load SVGs by icon name from a centralized registry

Customizable

Easily customize color, size, and container styles to match your design system

Lightweight

Zero dependencies, minimal footprint, and optimized performance

How to Use


<!-- Step 1: Add the script files to your HTML -->
<head>
  <!-- Include the SVG Icon Web Component scripts -->
  <script src="path/to/icons.js"></script>
  <script src="path/to/svg-icon.js"></script>
</head>

<!-- Step 2: Basic usage -->
<svg-icon name="home"></svg-icon>

<!-- Step 3: Customized icon with properties -->
<svg-icon 
    name="home" 
    color="#ff5722" 
    container_width="32px" 
    container_height="32px">
</svg-icon>
                    

// Step 1: Add scripts to angular.json
"scripts": [
    "path/to/icons.js",
    "path/to/svg-icon.js"
]

// Option A: Using NgModule
// app.module.ts
import { CUSTOM_ELEMENTS_SCHEMA, NgModule } from '@angular/core';

@NgModule({
    declarations: [...],
    imports: [...],
    schemas: [CUSTOM_ELEMENTS_SCHEMA],  // Allows using custom elements
})
export class AppModule {}

// Option B: Using Standalone Component
// my-component.ts
import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@Component({
    selector: 'app-my-component',
    standalone: true,
    imports: [...],
    schemas: [CUSTOM_ELEMENTS_SCHEMA],
})
                    

// In your main file (e.g., index.js or App.js)
import 'path/to/icons.js';
import 'path/to/svg-icon.js';

// In your component
function IconExample() {
  return (
    <div>
      <svg-icon name="home" color="#ff5722"></svg-icon>
    </div>
  );
}

// If using TypeScript, add the following to a .d.ts file:
declare namespace JSX {
  interface IntrinsicElements {
    'svg-icon': any;
  }
}
                    

// main.js
import 'path/to/icons.js';
import 'path/to/svg-icon.js';

// In .vue component template
<template>
  <div>
    <svg-icon name="home" color="#ff5722"></svg-icon>
  </div>
</template>

// Define in Vue config (main.js)
app.config.compilerOptions.isCustomElement = tag => tag.includes('-')

Component Properties

The SVG Icon Web Component accepts the following properties to customize its appearance and behavior:

Property Description Type Optional Default Value
name Name of the icon to render string No ''
color Fill color of the SVG paths string Yes '#000'
view Custom viewBox value string Yes ''
container_width Width of the icon container string Yes '20px'
container_height Height of the icon container string Yes '20px'
container_classes Extra CSS classes for the container string Yes ''

How to Add New Icons

1

Prepare Your SVG

Optimize your SVG file. For best results, export your icon from a design tool like Figma, Sketch, or Adobe Illustrator. 


<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24">
  <path d="M12 2L2 12h3v8h14v-8h3L12 2z" fill="currentColor"/>
</svg>
2

Add SVG File

Place your SVG file in the icons/ folder


icons/
  ├── home.svg
  ├── user.svg
  ├── settings.svg
  └── your-new-icon.svg
3

Register the Icon

Add an entry to the ICON_Files object in icons.js 


// Set the base path for icons
ICON_Files_BasePath = 'icons/'; // Base path for icons   
             
// In dist/icons.js
const ICON_Files = {
  home: 'home.svg',
  user: 'user.svg',
  settings: 'settings.svg',
  newIcon: 'your-new-icon.svg'  // Add your new icon here
};
4

Use Your New Icon

<svg-icon name="newIcon" color="#64b5f6"></svg-icon>

Best Practices

  • Keep SVGs simple with minimal paths for better performance
  • Use consistent viewBox sizes (e.g., 24x24) for all icons
  • Use semantically meaningful names for your icons
  • Use tools like SVGOMG to optimize your SVG files
